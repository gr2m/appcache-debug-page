===========================

[appCache is a douchbag](http://alistapart.com/article/application-cache-is-a-douchebag)

```html
<!DOCTYPE html>
<html>
<head>
  <title>appCache Test</title>
</head>
<body>
  <h1>appCache Test</h1>
</body>
</html>
